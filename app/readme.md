1. Rxjava有哪些优势?

具备响应式编程该有的特性
为异步而生, 无需手动创建线程, 并具备线程切换能力.
链式调用, 保证代码简洁
各种操作符, 功能非常强大, 满足各种业务需求.
简化异常处理.

2. Rxjava适用场景? 

网络请求, 数据库读写, 文件读写, 定时任务.. 等
各种耗时操作需要通过异步来完成的操作, 都可以使用Rxjava;

3. Rxjava 是一种特殊的观察者模式, 观察者, 被观察者, 订阅.
